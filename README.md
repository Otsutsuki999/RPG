# 🌑 Mundo Ninja RPG

Um site interativo de ranking de ninjas inspirado no universo Naruto. Explore o ranking dos shinobi, visualize informações de clãs e classificações.

## ✨ Características

- 🏆 **Ranking Dinâmico** - Visualize o ranking dos ninjas organizados por poder
- 🔍 **Busca Inteligente** - Procure ninjas por nome ou clã
- 👤 **Detalhes do Personagem** - Clique em qualquer ninja para ver mais informações
- 🎨 **Design Temático** - Interface inspirada no universo ninja com tema escuro e gradientes roxos
- 📱 **Responsivo** - Funciona perfeitamente em dispositivos móveis

## 🚀 Como Usar

1. Abra `index.html` no seu navegador
2. Use a barra de busca para encontrar ninjas específicos
3. Clique em qualquer linha da tabela para ver detalhes completos do personagem
